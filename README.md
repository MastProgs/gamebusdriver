---------------------------------
IOCP 서버는 이미 발로도 짤 수 있기 때문에, IOCP 서버는 내 github 계정 어딘가를 뒤져보다 보면 튀어나올 것이다.
서버는 boost asio 로 짜여졌고, Azure DB SQL 과 연동이 되며, 타이머 쓰레드 큐를 통하여 초당 이벤트 관리를 효율적으로 할 수 있다.
물론 AI 도 서버에서 자체적으로 훌륭하게 잘 관리하며 처리하고 있다.
