여기서부턴 우주초월이 임의로 만든 소스
================================
fork 를 눌러서, 자기 github 으로 프로젝트를 복사해 오게 된다.
  * 마음대로 수정해도, 일단 내꺼에만 저장하게 되고, 원본 전체 마스터 프로젝트에는 변동이 없다.
  * 이렇게 점도 찍을 수 있고

> 왼쪽 앞에 만들 수 있고
> 선이 주륵주륵

전부다 수정하고 왼쪽 위의 New Pull Request 를 눌러 원본에 이걸 적용해 달라고 요청한다.
원본 소유자는 이 요청을 적용하고 수정적용한다.
